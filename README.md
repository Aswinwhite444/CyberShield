# CyberShield 🔐

CyberShield is a web-based cybersecurity training platform designed to help users learn security concepts through interactive missions and gamification.

## Features
- User Login and Signup
- Avatar Selection
- Mission-Based Learning
- XP System
- Leaderboard
- Live Threat Feed

## Technologies Used
- Frontend: React (TypeScript)
- Backend: Node.js (Express)
- Database: MongoDB

## How to Run the Project

### Frontend
npm install  
npm run dev  

### Backend
cd backend  
npm install  
npm start
